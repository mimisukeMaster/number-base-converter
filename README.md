## description
任意の進数表記で示された数を、別の進数表記へ変換するpythonプログラムです。一度10進数に変換したのち、任意の進数表記へ変換しています。

10進数での数に対応する記号は[`ganerate_symbols.py`](gamerate_symbols.py)にて設定しています。
<br>現在、数字、大文字英、小文字英、平仮名、カタカナ、半角カナ、漢字が順に割り当てられています。
<br>記号の追加・変更が行えます。
